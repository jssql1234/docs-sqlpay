This release upgrades the database to version 149, improves access rights management, and resolves multiple bugs including payroll calculation errors, CP22 submission issues, and OT/leave record handling. (*238*)

<!-- truncate -->

## Database

Database upgraded to version **149**

---

## Updates

- Revised access right captions to use positive terminology
- Added **Administrative** group to access rights
- **Payroll Mobile Connect**:
  - Added PCB Table

---

## Bug Fixes

- Fixed `PR.EPF.BorangA.2009.Report` Grand Total displaying incorrect value
- Fixed month-end payroll incorrectly reducing total wages by amounts already processed in **Ad Hoc payroll** (#6066)
- Fixed missing prompt message when claim exceeds the limit (#6256)
- Fixed "Kesilapan format pada medan: No Kad Pengenalan Pasangan" error when submitting **CP22** file to IRBM (#6281)
- Fixed incorrect **Total Early Out** value after updating payroll for multi in/out logs
- Fixed missing function in cell C1 for **PB Enterprise ECP Payroll (Excel)** format (#6284)
- Fixed access rights not functioning for some data processors
- Fixed missing access rights for **Contract Stamp Duty** and **Print Stamp Duty Listing** (#6287)
- Fixed error log prompt when processing month-end for the current month
- Fixed incorrect **Total OT** value after updating payroll
- Fixed TMS summary showing **Time Off** and **Total Work = 0** instead of **On Time**
- Fixed **Apply** button on **Pending Payroll Listing** having no effect (#6283)
- Fixed inability to assign action to **OT** and **Leave** records
- Fixed "Abnormal Program Termination" on BCD default expression in Win64 (#7712)
