This release includes a new feature to display total TMS licenses used, various bug fixes for Sync Cloud, TMS log overwriting, and payment file generation, along with updates to license messages and project options. (*235*)

<!-- truncate -->

## Features

- Displayed total TMS licenses used by the employee

---

## Updates

- Renamed "Electric Vehicles" to "Electric Vehicle Charger"

---

## Bug Fixes

- Fixed issue where IsPending status remained True after `Sync Cloud`
- Fixed issue where an extra leave record was created during `Sync Cloud`
- Fixed TMS log overwriting when the first log is after 9am
- Fixed LogPhoto checkbox not disabling when SQL Drive is disabled, and not resetting to False when **ByGPS** is unchecked
- Fixed payment file generation ignoring grid view filters and sorting
- Fixed `Dynamic SQL Error` when generating Yearly Gross Pay report with Exclude Advance option checked
- Fixed Identification Number field not populating with passport number in Merchantrade Top-Up file for foreign workers
- Fixed inability to copy **Rest Day** from one employee to another
- Fixed missing bank code for GX Bank in **e-AmBiz** Bulk Payment file format
- Fixed "Value of MD5 is not valid" error during **e-AmBiz** Bulk Payment submission
