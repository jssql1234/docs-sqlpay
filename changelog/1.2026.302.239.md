This release upgrades the database to version 150, adds new bank file formats for **RHB** and **Deutsche Bank**, and fixes multiple issues in TMS and payroll reports. (*239*)

<!-- truncate -->

## Database

Database upgraded to version **150**

---

## Features

- Added multiple payroll process filtering to **Payroll Credit Bank** report (#6289)
- Added **RHB Bank BulkPay Instant Transfer** file format (#6290)
- Added **Deutsche Bank Payroll** file format (#6255)
- Added option to set clock method for days without session

---

## Updates

- Added **Advance Period** to **Maintain Leave Type**
- Added **Attachment** to **Maintain Leave Type**

---

## Bug Fixes

- Fixed "Compensation for loss of employment" and "Gratuity" columns displaying only the taxable amount in the **Income Tax EA** form (#6282)
- Fixed generating **Hong Leong Bank Connect First** excel file with DuitNow payment mode showing error "IBG bank code not found" when employee bank is "Merchantrade" (#6293)
- Fixed incorrect **Total Early Out** for multi in/out logs
- Fixed missing log after filtering employee and editing log
- Fixed incorrect value when generating calendar date from reminder
- Fixed double posting for `TA_Trans` records after recalculation
- Fixed inability to edit PostDate for OT records
- Fixed incorrect column sequence generation for leave balance
