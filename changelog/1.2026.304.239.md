This release resolves multiple issues with the **AmAccessBiz Payroll** bank file, fixes a **Credit Bank Report** access error, and addresses a performance hang in the **Yearly Individual** report for large databases. (*239*)

<!-- truncate -->

## Bug Fixes

- Fixed invalid employee tax numbers in **AmAccessBiz Payroll IRB** excel (#6301)
- Fixed submitting **AmAccessBiz Payroll** excel file to bank returning error "U99999 - An unexpected error has occurred" (#6300)
- Fixed clicking **Credit Bank Report** from **Payment Summary** form showing error "Interface not supported" (#6299)
- Fixed application hanging for over 30 minutes when applying **Yearly Individual** report on databases with more than 1 million records (#6298)
