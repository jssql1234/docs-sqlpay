## Location

### Android

**From SQL HRMS app:**

1. Turn on **_Location service_**

    ![android-location1](../../../static/img/integration/hrms/permission/android-location1.png)

2. ** Select **_'While using the app'_**

    ![android-location2](../../../static/img/integration/hrms/permission/android-location2.png)

3. Select **_'Update settings'_** and select **_'Allow all the time'_** | Return to SQL HRMS

    ![android-location3](../../../static/img/integration/hrms/permission/android-location3.png)

**From Device Settings:**

1. Navigate to App info > App permissions > Location and select **_'Allow all the time'_**

    ![android-location4](../../../static/img/integration/hrms/permission/android-location4.png)

### iOS

**From SQL HRMS app:**

1. Go to Privacy & Security > Location Service and toggle **_'Location Services'_** on, return to SQL HRMS

    ![ios-location1](../../../static/img/integration/hrms/permission/ios-location1.png)

2. Select **_'Allow While Using App'_**

    ![ios-location2](../../../static/img/integration/hrms/permission/ios-location2.png)

3. Select **_'Change to Always Allow'_**

    ![ios-location3](../../../static/img/integration/hrms/permission/ios-location3.png)

**From Device Settings:**

1. Navigate to Settings > Privacy & Security > Location Service and toggle **_'Location Services'_** on

    ![ios-location4](../../../static/img/integration/hrms/permission/ios-location4.png)

2. Next, go to Settings, search for **_'SQL HRMS'_** > Location and select **_'Always'_**

    ![ios-location5](../../../static/img/integration/hrms/permission/ios-location5.png)

**Content & Privacy Restrictions:**

1. Go to Settings > Screen Time > Content & Privacy Restrictions

    ![ios-location6](../../../static/img/integration/hrms/permission/ios-location6.png)

    :::info
    If you have disabled this feature previously, you can skip all of the following steps
    :::

2. Enter your Screen Time Password if you have previously set one

3. Search for **_'Location Services'_** and select **_'Allow Changes'_**. Toggle **_'Location Services'_** on

    ![ios-location7](../../../static/img/integration/hrms/permission/ios-location7.png)

## Notification

### Android

**From SQL HRMS app:**

1. Select **_'Allow'_**

    ![android-noti1](../../../static/img/integration/hrms/permission/android-noti1.png)

**From Device Settings:**

1. Go to App info > App permissions > Notifications and select **_'Show notifications'_**

    ![android-noti2](../../../static/img/integration/hrms/permission/android-noti2.png)

### iOS

**From SQL HRMS app:**

1. Select **_'Allow'_**

    ![ios-noti1](../../../static/img/integration/hrms/permission/ios-noti1.png)

**From Device Settings:**

1. Navigate to Settings, search for **_'SQL HRMS'_** > Notifications, toggle **_'Allow Notifications'_** on

    ![ios-noti2](../../../static/img/integration/hrms/permission/ios-noti2.png)

## Camera

### Android

**From SQL HRMS app:**

1. Select **_'While using the app'_**

    ![android-camera1](../../../static/img/integration/hrms/permission/android-camera1.png)

**From Device Settings:**

1. Go to App info > App permissions > Camera and select **_'Allow only while using the app'_**

    ![android-camera2](../../../static/img/integration/hrms/permission/android-camera2.png)

### iOS

**From SQL HRMS app:**

1. Select **_'Allow'_**

    ![ios-camera1](../../../static/img/integration/hrms/permission/ios-camera1.png)

**From Device Settings:**

1. Navigate to Settings, search for **_'SQL HRMS'_** > Camera, toggle **_'Camera'_** on

    ![ios-camera2](../../../static/img/integration/hrms/permission/ios-camera2.png)

## Photo Library

### iOS

**From SQL HRMS app:**

1. Select **_'Allow Full Access'_**

    ![ios-photo1](../../../static/img/integration/hrms/permission/ios-photo1.png)

**From Device Settings:**

1. Navigate to Settings, search for **_'SQL HRMS'_** > Photos and select **_'Full Access'_**

    ![ios-photo2](../../../static/img/integration/hrms/permission/ios-photo2.png)

2. Select **_'Allow Full Access'_**

    ![ios-photo3](../../../static/img/integration/hrms/permission/ios-photo3.png)

## Nearby Devices / Bluetooth

### Android

**From SQL HRMS app:**

:::info
For **Android 11 and below**, may skip to Step 2
:::

1. Select **_'Allow'_**

    ![android-bluetooth1](../../../static/img/integration/hrms/permission/android-bluetooth1.png)

2. Turn on **_Bluetooth service_**

    ![android-bluetooth2](../../../static/img/integration/hrms/permission/android-bluetooth2.png)

**From Device Settings:**

:::info
Only for **Android 12 and above**
:::

1. Navigate to App Info > App permissions > Nearby devices and select **_'Allow'_**

    ![android-bluetooth3](../../../static/img/integration/hrms/permission/android-bluetooth3.png)

### iOS

**From SQL HRMS app:**

1. Select **_'Allow'_**

    ![ios-bluetooth1](../../../static/img/integration/hrms/permission/ios-bluetooth1.png)

2. Select **_'Settings'_**, toggle **_'Bluetooth'_** on and return to SQL HRMS

    ![ios-bluetooth2](../../../static/img/integration/hrms/permission/ios-bluetooth2.png)

**From Device Settings:**

1. Navigate to Settings, search for **_'SQL HRMS'_** and toggle **_'Bluetooth'_** on

    ![ios-bluetooth3](../../../static/img/integration/hrms/permission/ios-bluetooth3.png)

2. Navigate to Settings > Bluetooth and toggle **_'Bluetooth'_** on

    ![ios-bluetooth4](../../../static/img/integration/hrms/permission/ios-bluetooth4.png)
