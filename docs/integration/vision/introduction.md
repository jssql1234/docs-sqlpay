## What is SQL Vision

SQL Vision is a task management system designed to work with SQL Payroll & SQL Account. With SQL Vision, managers and employees share clear goals, track performance in real time, and get rewards transparently.

Find out more about [SQL Vision](https://www.sql.com.my/sql-vision/)

## Demo

This system supports both **mobile** and **web applications**, allowing users to access the platform across multiple devices.

The **mobile application** is designed for both **users and managers**, providing convenient access to essential features and enabling quick interactions on the go.

The **web application**, on the other hand, focuses more on **management and administrative functionalities**. It provides managers with a more comprehensive interface for monitoring, managing data, and performing administrative tasks.
